# 🚗 EASY DRIVE Autoškola s.r.o.

Moderní, responzivní multipage webová prezentace pro autoškolu vytvořená pomocí **HTML5, Tailwind CSS a Vanilla JavaScriptu**.

---

## 📌 O projektu

Tento projekt představuje profesionální web pro autoškolu zaměřený na:

* vysokou konverzi (CTA prvky)
* důvěryhodnost (reference, moderní vozy)
* jednoduchou orientaci uživatele
* moderní design (tmavý režim + akcentní žlutá)

---

## 🧱 Použité technologie

* **HTML5**
* **CSS3**
* **Tailwind CSS**
* **Vanilla JavaScript**

---

## 📁 Struktura projektu

```
easy-drive/
│
├── index.html        # Hlavní stránka
├── cenik.html        # Ceník
├── sluzby.html       # Služby
├── kontakt.html      # Kontakt
│
├── css/
│   └── style.css     # Vlastní styly
│
└── js/
    └── script.js     # Interaktivita a validace
```

## 🎨 Design systém
